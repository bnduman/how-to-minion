# Adding Images and Files

This guide covers how to upload images and files (like PDFs) to Plone and insert them into your pages.

---

## Uploading an Image

Before you can use an image on a page, you need to upload it to Plone.

### Step 1: Navigate to the Right Folder

Go to the section of the website where the image belongs. For example, if it's for a news item in the News section, navigate there first.

### Step 2: Upload

1. Click **Add new...** in the green toolbar.
2. Select **Image**.
3. You'll see a form with the following fields:
   - **Title:** Give the image a clear name (e.g. "Summer School 2026 Group Photo").
   - **Image file:** Click **Browse** or **Choose File** and select the image from your computer.
   - **Copyright:** Enter the copyright information for this image.
4. Click **Save**.

> ⚠️ **Always fill in the copyright field.** An image with missing copyright information can break the page or the content. If you're unsure about the source, check with the person who provided the image before uploading.

---

## Inserting an Image into a Page

Once the image is uploaded, you can insert it into any page:

1. Open the page in **Edit** mode.
2. Place your cursor where you want the image to appear.
3. Click the **Insert Image** button in the text editor toolbar (it usually looks like a small picture icon).
4. Browse to find the image you uploaded.
5. Select it and click **OK** or **Insert**.

## Alt Text

It is not required, but good practice, to add alt text to your images. Alt text is a short description that appears when an image can't load, and it helps with accessibility (e.g. screen readers for visually impaired visitors).

**Good alt text:** "Participants at the 2025 BCDSS annual conference"
**Bad alt text:** "image1" or leaving it empty

You can usually add alt text in the image insert dialog or in the image's properties.

---

## Uploading a File (PDF, Word Document, etc.)

The process for files is similar to images:

1. Navigate to the folder where the file belongs.
2. Click **Add new...** in the green toolbar.
3. Select **File**.
4. Fill in the form:
   - **Title:** A clear, descriptive name (e.g. "Summer School 2026 Programme").
   - **File:** Click **Browse** or **Choose File** and select the file from your computer.
5. Click **Save**.

To link the file from a page, see [Adding Links](adding-links.md).

---

## Image Best Practices

- **Use the right file format.** JPG for photos, PNG for graphics with transparency. Avoid BMP or TIFF.
- **Keep file sizes reasonable.** Very large images slow down the page. Resize images before uploading if they're wider than 2000 pixels.
- **Name your files clearly** before uploading. "summer-school-2026.jpg" is better than "IMG_4382.jpg."
- **Always include copyright information.** This is not optional. Missing copyright info can break the page.
