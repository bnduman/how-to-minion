# What Is Plone?

Plone is the software that runs the BCDSS website. It's where you go to add, edit, or remove anything on the site. No coding needed. It has a visual editor similar to Word.

## What you'll use it for

- Creating and editing pages
- Adding news items and events
- Uploading images, PDFs, and other files

# Finding Your Way Around

## The Toolbar on the Left

The toolbar at the left of the page is your main control panel. Here's what the key buttons do:

**Contents**
Shows everything inside the current folder (pages, images, files). Think of it as a file explorer.

**Edit**
Opens the current page in editing mode so you can make changes.

**Add new...**
Lets you create new content: a page, a news item, an event, an image, a file, or a folder.

**State**
Shows the publishing status of the current page (draft, published, pending review). This is where you make content go live.

**Display**
Controls how the page looks to visitors. You'll rarely need this.

**Actions**
A dropdown with the following options:

| Action | What It Does |
|---|---|
| Cut / Copy | Move or duplicate a page to another location in the site. |
| Delete | Permanently removes the page. This cannot be undone. |
| Rename | Changes the page's internal ID and URL. Use this if the page was created with the wrong name. |
| URL Management | Set up redirects, useful if a page has moved and you don't want old links to break. |
| Rich Results | Controls structured data for search engines. You'll rarely need this. |

---

## The Contents View

Click **Contents** in the toolbar to see everything inside the current section. This is one of the most useful views in Plone. It shows you:

- All pages, images, files, and subfolders in the current location.
- The publishing state of each item (draft, published, etc.).
- Options to rename, move, copy, or delete items.

Think of it like looking inside a folder on your computer.


---

## How Pages Are Organized

The BCDSS website is structured like a set of nested folders:

```
Home
├── About
│   ├── Team
│   └── Mission
├── Research
│   ├── Projects
│   └── Publications
├── News
└── Events
```

Every page lives inside a folder. When you create a new page, it goes into whatever folder you're currently in. This is important to keep in mind: always navigate to the correct section **before** you create something new.
